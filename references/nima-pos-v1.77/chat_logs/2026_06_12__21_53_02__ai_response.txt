أهلاً بك زميلي العزيز. بصفتي الخبير الهندسي ومصمم واجهات المستخدم (UI/UX) للمنصة، يسعدني جداً تقديم هذه الدراسة التحليلية الدقيقة والتوجيهات الهيكلية لمعالجة وتجميل شكل شاشة "طباعة وتصميم المنيو" للوصول إلى أعلى درجات الإتقان البصري العريق المتناسب مع نمط Modern Light Theme والأعراف الطباعية الراقية:

---

### 1. إصلاح الهيكل ومشاكل التكرار (Layout & Data Auditing)

* **حل تكرار المنتجات في القسم (Preventing Duplicate Render Rows):**
  * **السبب الهندسي:** ينشأ هذا التكرار عندما تحتوي مصفوفة المنتجات المسترجعة من قاعدة البيانات (`db.products`) على سجلات مكررة أو عند تكرار استدعاء فلترة الفئات بشكل متداخل داخل المصفوفة المُسطحة (`allItems`).
  * **التوجيه البرمجي الموصى به:** بدلاً من سحب كافة العناصر بشكل مباشر، سنقوم بتطبيق فلتر تصفية فريد (De-duplication layer) باستخدام الـ `id` أو `name` الخاص بكل منتج قبل تسطيح المصفوفة وعرضها في `pagesList`.
  * **المقترح التقني:**
    ```typescript
    // تمرير مصفوفة العناصر عبر مُرشح فريد باستخدام Set
    const seenIds = new Set();
    const uniqueCategoryItems = availableProducts.filter(p => {
        if (seenIds.has(p.id)) return false;
        seenIds.add(p.id);
        return p.category === cat;
    });
    ```

* **ضبط محاذاة العناوين الرئيسية والفواصل العلوية:**
  * **السبب الهندسي:** تخطيط الأعمدة المتعددة (`columns-2`) في المتصفحات يوزع العناصر عمودياً بشكل تلقائي. إذا صادف وجود عنوان فئة في نهاية العمود الأول، فإنه ينقسم أو يزاح بصورة حادة. تداخل العناوين مع الخط الفاصل يعود لعدم كفاية مساحة الحماية (Margin Tunnels).
  * **التوجيه البرمجي الموصى به:**
    1. إسناد خاصية `break-inside: avoid-column` و `display: inline-block` أو `display: block` مع تعطيل الـ flex الرأسي للقسم الخارجي لكل عنوان فئة لمنعه تماماً من الانقسام بين عمودين.
    2. زيادة المساحة العلوية لعناوين الأقسام (`mt-8` بدلاً من `mt-4`) وإضافة إطار حماية بيضاء خلف النص لمنع امتداد الخطوط المنقطة تحته بصرياً.

---

### 2. تحسين التوزيع البصري وقابلية القراءة (Typography & Alignment)

* **محاذاة الأسعار والخطوط المنقطة (Price Dots Connector Stabilization):**
  * **السبب الهندسي:** يرجع التصاق السعر بالنقاط المنقطة أو عدم انتظامه الرأسي إلى استخدام عناصر مطلقة المواضع وغير مرنة الاتساع.
  * **التوجيه البرمجي الموصى به:**
    * إعادة هيكلة سطر اسم الطبق والسعر برمجياً ليعتمد على توزيع `flex justify-between items-baseline w-full`.
    * إعطاء حقل العنوان تظليلاً خلفياً بلون الخلفية المعينة (مثل `bg-[#FAF8F5]` في النمط الكلاسيكي) مع حماية أفقية (`pl-2`).
    * جعل سطر النقاط المنقطة يمتد كعنصر تالٍ ذو مرونة عالية وبخلفية خلفية شفافة، ووضع السعر في حاوية مستقلة جهة اليسار بـ `pr-2` وخلفية ممتصة للضوء تمنع تداخل النقاط مع الرقم والعملة.

* **توحيد اتجاه العناصر والأيقونات هندسياً (RTL Harmony):**
  * **المشكلة الحالية:** تفاوت قراءة الأعمدة؛ حيث ينقاد التخطيط لعناصر مرنة تنظر للاتجاه العادي والـ layout العكسي تارة أخرى دون الالتزام التام بقواعد القراءة العربية.
  * **التوجيه الهندسي المقترح:**
    * فرض اتجاه موحد تماماً لكرت الوجبة (`renderMenuItem`) وجعله يبدأ دائماً بالصورة/الأيقونة على اليمين لتمثل مرساة العين (Eye-anchor)، تليها المساحة النصية للبيانات التفصيلية للمنتج ومكوناته، وتتجه السلسلة أفقياً لتنتهي دائماً بالسعر بوزنه العريض في أقصى اليسار.
    * كتابة كلاسات المحاذاة أفقياً بـ `text-right` للمسمى والوصف، و `text-left` للسعر في نهاية الممر.

---

### 3. اللمسات الجمالية وهوية البراند (Brand UI Refinement)

* **شعار الهوية العلوية ومساحة التنفس (Top Hero Banner Spacing):**
  * **التوجيه البرمجي الموصى به:** الإطار الذهبي الكلاسيكي المرسوم في الخلفية يمتلك هوامش ثابتة (`inset-3` / `inset-4`). لضمان تلافي اللمس المباشر للأيقونة للشريط الذهبي، يجب إضافة حقل حشوة علوية أماني (`pt-10` أو `pt-12`) لكتلة الهيدر الأوّل في الصفحة الأوّلى، ما يخلق فراغاً سماوياً يسمى "مساحة التنفس الفاخرة" (Luxury Breath Space).

* **ترشيق خطوط المكونات والوصف (Visual Hierarchy Scaling):**
  * **التوجيه البرمجي الموصى به:** لتسهيل المسح البصري (Scannability)، سنقوم بتباين الأوزان والألوان:
    * اسم الصنف الرئيسي: خط غامق عريض (`font-extrabold` / `font-bold` بوزن 800) وبلون غامق صلب.
    * سطر الوصف: خط نحيف خفيف (`font-light` بوزن 300 أو `font-normal` بوزن 400)، مع تصغير حجم الخط إلى (`text-[12px]`) واستخدام لون رمادي دافئ هادئ وشفافية لطيفة (`opacity-70` أو `text-slate-500`) لتوفير التدرج الممتع والمريح للعين.
