# Nima Frontend Starter

Starter web frontend for future Nima sales applications.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Current Starter Screens

- Dashboard
- POS
- Products
- Customers
- Orders
- Reports
- Settings

## Extension Direction

- Add more routes through `src/app/router.tsx`
- Add mode-aware navigation through `src/app/config/navigation.tsx`
- Promote shared page patterns into reusable components under `src/components`
- Expand from `starter` to `standard`, `service`, or `enterprise` using `src/app/config/appMode.ts`
